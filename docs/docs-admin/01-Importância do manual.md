# Importância do manual

Ler esta documentação (manual) é fundamental para entender o funcionamento do CACiC Event Manager, para aprender a utilizá-lo de forma eficiente e para conhecer as melhores práticas de uso do aplicativo.

Ao realizar operações no CACiC Event Manager sem seguir as orientações do manual, o usuário pode cometer erros que podem resultar em consequências como **perda permanente de dados**.

Se, ao final da leitura, o administrador ainda tiver dúvidas, ele pode entrar em contato com a equipe de desenvolvimento do CACiC Event Manager, que está disponível para ajudar e esclarecer quaisquer questões.
