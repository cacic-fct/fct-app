---
title: Fornecimento de dados para ministrantes
---

Organizadores podem solicitar o fornecimento de uma planilha (.csv) com os seguintes dados dos participantes de um evento:

- Nome;
- E-mail;
- Vínculo;
  - RA.
- Data da inscrição.

Não serão fornecidos dados como uuid, CPF e número de celular.
